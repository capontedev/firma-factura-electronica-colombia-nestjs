## Descripción

Servicio desarrollado en [NestJS](https://nestjs.com) y [xadesjs](https://xadesjs.com/) para el firmado facturas electrónicas DIAN Colombia.

## Instalación

```bash
$ npm install
```

## Ejecutar App

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# documentación 
$ http://localhost:3000
```
